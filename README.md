Inventory & Supply Chain Management Dashboard

This project presents a Power BI dashboard analyzing inventory performance and warehouse efficiency across product categories and locations. The goal is to monitor key supply chain metrics and identify patterns that impact operational performance.

⸻

Business Objective

Provide a clear view of how inventory moves through warehouses and highlight inefficiencies such as underutilized storage or slow-moving inventory.

⸻

Key Metrics

• Warehouse Utilization Rate – Percentage of warehouse capacity currently used
• Days Sales of Inventory (DSI) – Average number of days required to sell inventory
• Inventory Turnover Ratio – Frequency at which inventory is sold and replenished

⸻

Tools Used

Power BI
Power Query
DAX
